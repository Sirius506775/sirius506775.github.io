백엔드 공부 시에 항시 운영되는 외부 서버와의 통신 환경과 서버 구축 단계를 학습! 하기 위해 라즈베리 파이를 구매햇다. 라즈베리파이에 우분투 서버를 설치해보고자 한다.

제품 구성 :

  • Raspberry Pi 4 Model B 4GB RAM
  • SD Card 64GB

1. sd card format

운영체제를 설치하기 전에 sd card format을 진행한다.

라즈베리파이의 부트로더는 FAT16, FAT32 등의 FAT 파일 시스템을 사용하기 때문에 format을 두 번 진행한다.

1.1 sd card Formatter 설치 및 실행

SD Association 사이트에 접속해서 다운로드한다. 필자는 windows 환경임

1.2 sd카드를 인식하고, formatter를 실행한다.

포맷할 sd 카드 경로를 지정하고 포맷한다.

포맷 완료 화면에서 File System이 FAT32라면 그대로 진행해도 되지만, 만약 ’exFAT’인 경우는 변경을 해줘야한다.

FAT32 Format으로 File System을 FAT32로 변경한다.

Volume label을 적어주지 않으면 에러가 발생할 수 있다.

파일 시스템을 확인해보면 FAT32 형식으로 변경되어있는 것을 확인할 수 있다.
